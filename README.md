## Marks calculator for Ekool.eu
Calculator for calculating average mark quickly.

## Instruction
1. login into ekool.eu
2.  Open your profile
3. open "Marks" tab
4. Right click with your mouse
5. Select "Save as"
6. Save .html file
7. in a programm press "open file"
8. Choose the file that you have saved
